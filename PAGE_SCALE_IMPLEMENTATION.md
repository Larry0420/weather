# 頁面縮放功能實作說明

## 功能概述

已在設定頁面的無障礙功能區塊中添加了頁面縮放功能，讓使用者可以調整整個應用程式的縮放比例，提升可讀性和使用體驗。

## 實作內容

### 1. AccessibilityProvider 擴展

在 `AccessibilityProvider` 類別中添加了以下功能：

- **新增屬性**：
  - `_pageScale`: 頁面縮放比例（預設值：1.0）
  - `pageScale` getter: 獲取當前頁面縮放比例

- **新增方法**：
  - `setPageScale(double scale)`: 設定頁面縮放比例（範圍：0.8-2.0）
  - `getPageScaleLabel(double scale, bool isEnglish)`: 獲取縮放比例的語言標籤

- **新增常數**：
  - `_pageScaleKey`: SharedPreferences 儲存鍵值
  - `pageScaleOptions`: 預設縮放選項 [0.8, 0.9, 1.0, 1.2, 1.5, 2.0]

### 2. 語言支援

在 `LanguageProvider` 中添加了：
- `pageScale`: 頁面縮放的語言標籤（英文：'Page Scale'，中文：'頁面縮放'）

### 3. 設定頁面 UI

在設定頁面的無障礙功能區塊中添加了：

- **頁面縮放卡片**：顯示當前縮放設定
- **滑桿控制**：可拖曳調整縮放比例（0.8-2.0）
- **快速選項**：提供預設縮放比例的選擇按鈕
- **視覺指示器**：使用縮小/放大圖示作為滑桿端點

### 4. 全域縮放應用

在 `MaterialApp` 中使用 `MediaQuery` 應用頁面縮放：

```dart
MediaQuery(
  data: MediaQuery.of(context).copyWith(
    textScaleFactor: accessibility.pageScale,
  ),
  child: MaterialApp(...)
)
```

## 功能特點

### 縮放範圍
- **最小值**：0.8（縮小 20%）
- **最大值**：2.0（放大 100%）
- **預設值**：1.0（正常大小）

### 預設選項
- 非常小 (0.8)
- 小 (0.9)
- 正常 (1.0)
- 大 (1.2)
- 非常大 (1.5)
- 超大 (2.0)

### 雙語支援
- 英文：Very Small, Small, Normal, Large, Very Large, Extra Large
- 中文：非常小、小、正常、大、非常大、超大

### 持久化儲存
- 使用 SharedPreferences 儲存使用者設定
- 應用程式重啟後自動恢復上次的縮放設定

## 使用方式

1. 進入設定頁面
2. 在「輔助功能」區塊中找到「頁面縮放」
3. 使用滑桿調整縮放比例
4. 或點擊預設選項快速設定
5. 設定會立即生效並自動儲存

## 技術實作

### 縮放機制
使用 Flutter 的 `MediaQuery.textScaleFactor` 來實現全域縮放，這會影響：
- 所有文字大小
- 按鈕和輸入框
- 圖示大小（配合現有的圖示縮放）
- 間距和佈局

### 響應式設計
- 橫向和直向模式都支援
- 與現有的文字大小和圖示大小設定協調工作
- 保持 UI 元素的比例和對齊

### 效能優化
- 縮放設定變更時只觸發必要的重建
- 使用 Provider 模式確保狀態管理效率
- 避免不必要的重新計算
