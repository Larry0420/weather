# 路線頁面選擇器邊框優化

## 🎯 優化目標

優化路線頁面選擇器中標籤之間的邊框和視覺分隔，提升界面的美觀性、層次感和用戶體驗，同時保持緊湊的佈局設計。

## ✨ 主要優化內容

### 1. 視覺分隔線優化

#### 橫向模式
- **垂直分隔線**：在地區選擇器和路線選擇器之間添加 1px 寬度的垂直分隔線
- **分隔線樣式**：使用主題的 outline 顏色，透明度為 0.12
- **分隔線高度**：固定高度 80px，確保視覺一致性
- **分隔線邊距**：左右邊距 8px，與容器內邊距保持一致

#### 直向模式
- **水平分隔線**：在地區選擇器和路線選擇器之間添加 1px 高度的水平分隔線
- **分隔線樣式**：使用主題的 outline 顏色，透明度為 0.12
- **分隔線邊距**：上下邊距 8px，與卡片間距保持一致

### 2. 卡片式佈局優化

#### 橫向模式
- **地區選擇器卡片**：獨立容器，圓角 8px，內邊距 8px
- **路線選擇器卡片**：獨立容器，圓角 8px，內邊距 8px
- **卡片邊框**：使用主題 outline 顏色，透明度為 0.06，寬度 0.5px

#### 直向模式
- **地區選擇器卡片**：獨立容器，圓角 8px，內邊距 10px
- **路線選擇器卡片**：獨立容器，圓角 8px，內邊距 10px
- **卡片邊框**：使用主題 outline 顏色，透明度為 0.06，寬度 0.5px

### 3. 圖標和標籤優化

#### 圖標設計
- **地區圖標**：使用 `Icons.location_city_outlined`，顏色為主題 primary 色
- **路線圖標**：使用 `Icons.route_outlined`，顏色為主題 secondary 色
- **圖標大小**：16 * accessibility.iconScale，適配無障礙設置
- **圖標間距**：與標籤文字間距 6px

#### 標籤樣式
- **字體大小**：13 * accessibility.textScale，保持一致性
- **字體粗細**：FontWeight.w600，突出重要性
- **顏色**：使用主題的 onSurface 顏色，確保對比度

### 4. 間距和佈局優化

#### 橫向模式
- **卡片間距**：8px 內邊距，確保內容不會過於擁擠
- **標籤與內容間距**：6px，保持視覺層次
- **分隔線邊距**：8px 左右邊距，與容器內邊距一致

#### 直向模式
- **卡片間距**：10px 內邊距，提供更舒適的觸控體驗
- **標籤與內容間距**：6px，保持視覺層次
- **分隔線邊距**：8px 上下邊距，與卡片間距一致

## 🔧 技術實現

### 1. 方法重構
- 創建 `_buildOptimizedLandscapeRouteSelector()` 方法
- 創建 `_buildOptimizedPortraitRouteSelector()` 方法
- 保持原有方法以備後續優化

### 2. 容器優化
- 使用 `Container` 包裝每個選擇器區域
- 統一的 `BoxDecoration` 樣式
- 一致的邊框和圓角設計

### 3. 分隔線實現
- 使用 `Container` 創建分隔線
- 動態高度/寬度適配不同模式
- 主題顏色適配，支持深色/淺色模式

### 4. 響應式設計
- 橫向模式：並排佈局，垂直分隔線
- 直向模式：垂直佈局，水平分隔線
- 自適應間距和邊距

## ✅ 總結

通過這次邊框優化，路線頁面選擇器在保持緊湊佈局的同時，顯著提升了視覺美觀性和用戶體驗。主要改進包括：

1. **視覺優化**：添加分隔線和卡片式佈局，提升界面層次感
2. **結構優化**：重新設計佈局結構，使功能區分更加清晰
3. **樣式優化**：統一邊框樣式，提升設計一致性
4. **體驗優化**：圖標和標籤的優化，提升操作直觀性

這些優化為用戶提供了更美觀、更易用的界面，同時保持了原有的緊湊設計理念。新的邊框設計不僅提升了視覺效果，還增強了功能區分的清晰度，為後續的界面優化奠定了良好基礎。
